# CampusMetrics Dashboard: Student Performance Analytics using Power BI and Tableau

## Overview

CampusMetrics Dashboard is an academic analytics project developed as a Semester Project-II at R. C. Patel Institute of Technology, Shirpur. The project transforms raw student academic records into interactive and insightful dashboards using Power BI and Tableau.

The dashboard helps analyze student performance, attendance, grades, CGPA, SGPA, and examination results through visual reports that support data-driven academic decision-making.

## Problem Statement

Educational institutions generate large volumes of academic data every semester. Analyzing this information manually is time-consuming and prone to errors. This project aims to simplify academic performance monitoring through interactive visualizations and dashboards.

## Dataset

The dashboard is built using a college academic result dataset containing:

* Student Registration Number
* Department / Branch
* Semester Information
* CGPA
* SGPA
* Attendance Marks
* Internal Assessment Scores
* End Semester Marks
* Grade Information
* Result Status (Pass/Fail)

### Power BI Dashboard

* Student Performance Overview

* Grade Distribution
* Attendance Analysis

* Interactive Filters and Slicers

### Tableau Dashboard

* Average CGPA Analysis
* Average SGPA Analysis
* Department-wise Performance
* Pass/Fail Analysis
* Academic Category Analysis
* Interactive Visual Reports

## Key Insights

* Track semester-wise academic performance.
* Compare department-wise student outcomes.
* Analyze grade distribution patterns.
* Monitor attendance and examination performance.
* Identify performance trends across semesters.
